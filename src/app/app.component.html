TOMa

Application for the automated navigation, inspection and analysis of the Testable Object Model

Current or Intended Product Features

  • All testable objects represented in graph database
  • Selectors, paths and configuration separated from code
  • Tests generated dynamically from user parameters
  • Built on Selenium and much of the existing framework
  • Web client lowers learning curve for testers & speeds up test creation
  • Unique entity ID's makes test coverage much easier to calculate
  • Test grid also represented in graph database
  • Load testing is easier to set up and administer
  • Application can relaunch tests after correcting for certain failures
  • Real-time test results accessible through web and mobile clients
  • Tests can be launched and results retrieved through REST API
  • Parsing of testable objects in HTML source can be automated
  • Application can be integrated with Jira to simplify test management
  • No additional costs to USANA, no additional hires
  • Application is commercializable -- nothing like it is currently available
